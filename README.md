# Mindset Jukebox

Radio-quality songs that teach you how the world's smartest people think. Built by [TJ Larkin](https://tjlarkin.com).

## Stack
- Static HTML/CSS/JS (no framework)
- Audio hosted on GitHub via [SFYS-music repo](https://github.com/tjlark23/SFYS-music)
- Deployed on Cloudflare Pages

## Structure
- `index.html` — Homepage
- `songs.js` — Song & people data
- `tj-headshot.png` — Footer photo
- `_redirects` — Cloudflare Pages routing
